# HƯỚNG DẪN SỬ DỤNG

## Yêu cầu
- Đã cài đặt **Node.js**
- File Google Docs phải ở chế độ **View only**
- Chỉ hỗ trợ **tải định dạng Google Docs (.doc)**

---

## Các bước thực hiện

### Bước 1: Nhập link tài liệu
- Mở file `inp.txt` trong thư mục `input`
- Dán **link Google Drive hoặc Google Docs (view only)** vào file  
- Mỗi dòng chứa **1 link**

Ví dụ:  
https://drive.google.com/file/d/xxxxxx/view  
https://docs.google.com/document/d/xxxxxxx/view

---

### Bước 2: Chạy chương trình
Mở **Terminal / Command Prompt** tại thư mục dự án và chạy lệnh:

```bash
node main.js
